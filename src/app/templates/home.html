Welcome to SNieves-Doñana Benchmark

A benchmark for animal detection models using images from various parks.

About this project

Wildlife image classification and detection represent key technologies for monitoring animal populations, protecting endangered species, and identifying predatory species that impact livestock using camera traps.

This project (inspired by the Serengeti Benchmark, focused on the fauna of Serengeti National Park in Tanzania, Africa) will be implemented in the Doñana and Sierra de las Nieves parks. More than 30 terabytes of images provided by the EnBiC2Lab Lab project will be used.

Scientific comparison is essential. By establishing a benchmark, a standard is provided against which other researchers can evaluate their results, fostering innovation in the field.

Our objective

By developing a benchmark using images from these parks and providing the dataset. Our goal is to encourage other researchers to develop their own models with this data and compare them with ours.

Features

The functionalities developed in this project include:

  • An interface for downloading and filtering over 30 terabytes of camera trap images collected from two protected areas.
  • An interface for applying an animal detection model.
  • An API enabling programmatic access to the previously mentioned functionalities.
