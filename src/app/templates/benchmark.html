The Benchmark

The benchmark is the calculation of the following metrics obtained by testing the model with the test dataset:

  • Accuracy
  • Precision
  • Recall
  • F1 Score
  • Average IoU (Intersection over Union)

This testing script was the one used for benchmarking the models, its output was then used to calculate the metrics using this notebook.

The Models

The benchmarked models are:

  • YOLO
  • Megadetector
  • FasterRCNN

These models were trained and validated with the same dataset found in the dataset page.
